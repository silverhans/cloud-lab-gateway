# Task B5.1 — SSE broker (live lab / deploy / check / quota events)

You are a coding agent on **Cloud Lab Gateway**. Read `CLAUDE.md` and
`docs/ARCHITECTURE.md` first.

## Goal

Implement the Server-Sent Events broker so the browser gets realtime updates:
`internal/adapters/sse/` + a `GET /sse/labs` endpoint, wired into `cmd/gateway`.

## Reference — read first

- `internal/ports/system.go` — `SSEBroker` interface (implement it).
- `internal/ports/queue.go` — `EventBus` (the broker subscribes to it).
- `internal/adapters/queue/asynq/eventbus.go` — the Redis pub/sub `EventBus`.
- `internal/app/outbox/publisher.go` — domain events flow outbox → `EventBus`.
- `internal/domain/lab/events.go` — the domain event kinds.
- `web/src/lib/sse.ts` — the frontend's expected event shapes (**the contract**).

  is mounted at the root, **outside** `/api/v1` and the strict server.

## Event contract — must match web/src/lib/sse.ts

Each SSE `data:` line is JSON, discriminated on `type`:

- `lab.state_changed` — `{labId, state, reason}`
- `lab.deploy_progress` — `{labId, step, status}`
- `check.state_changed` — `{labId, checkId, state}`
- `quota.snapshot` — `{cpu, ram, disk}`

The broker maps domain/outbox events to this shape. If a domain event has no
SSE projection, drop it.

## Behaviour

- `GET /sse/labs` — `Content-Type: text/event-stream`, requires the
  `clg_session` cookie (reuse the httpapi session validation). Streams events
  for **the authenticated user's own labs**; a teacher/admin scope can come
  later — keep this task to owner-scoped filtering.
- The broker subscribes to the `EventBus` once, fans out to all connected
  clients. One goroutine per connection; close on `request.Context().Done()`.
- Bounded per-client buffer — a slow client is dropped, never blocks the bus.
- Periodic keepalive comment (`: ping\n\n`) every ~20s so proxies hold the
  connection.
- Flush after every write (`http.Flusher`).
- `GetStreamLabs` in the generated strict server stays `501` — the real stream
  is `/sse/labs`, consumed directly by the browser (not via the typed client).
