## Detailed specs

### `internal/app/lab/deps.go`

```go
type Deps struct {
    UoW          ports.UnitOfWork
    Pool         ports.PoolRepo
    Lab          ports.LabRepo
    Courses      ports.CourseRepo
    Audit        ports.AuditRepo
    QuotaCache   QuotaCache
    Cloud        ports.CloudProvider     // only for fallback fetch when cache empty
    Queue        ports.TaskQueue
    Clock        ports.Clock
    Settings     ports.SettingsRepo
    Logger       *zap.Logger
}

// QuotaCache wraps Redis-backed (or DB-backed) snapshot store.
type QuotaCache interface {
    Read(ctx) (snap shared.QuotaSnapshot, age time.Duration, err error)
    Write(ctx, snap shared.QuotaSnapshot) error
}
```

### `internal/app/lab/create.go`

```go
type CreateInput struct {
    StudentUserID  shared.UserID
    CourseID       shared.CourseID
    LabTemplateID  shared.LabTemplateID
    RequestID      string  // from middleware, for audit correlation
}

func (d Deps) CreateLab(ctx context.Context, in CreateInput) (*lab.LabInstance, error)
```

Algorithm in plain prose (commit this exact sequence; deviation = bug):

1. Resolve `quota.threshold_pct` from `SettingsRepo` (with fallback to `quota.DefaultThresholdPct`).
2. Resolve the resource profile for `LabTemplateID`.
   For this PR, do **not** change `internal/ports/` to invent a new
   `LabTemplateRepo`. Implement a small internal helper in
   `internal/app/lab/create.go` (or a sibling file) that maps known
   `LabTemplateID` values to `shared.ResourceRequest`, and leave a clear
   `TODO` noting that this should later come from persisted lab-template
   topology.
3. Compute `req := resourceRequestFor(in.LabTemplateID)`.
4. Read `QuotaCache`:
    - If `age > 2 * 30s` → enqueue `TaskRefreshQuota`, return `shared.ErrCloudUnavailable` (HTTP layer maps to 503).
5. Call `quota.Evaluate(snap, req, threshold)`. If `!Allowed`:
    - Append `audit.AuditEvent{Kind: KindQuotaBlocked, Payload: {snap, req, predicted_pct}}`.
    - Return `shared.ErrQuotaExceeded`.
6. Begin tx (`UoW.WithTx`):
    - Resolve the course and its `KIDomainID` through `Courses` (consume
      `ports.CourseRepo`; do not add raw SQL to the use case and do not
      change `internal/ports/`).
    - Check `LabRepo.FindActiveByStudentAndCourse` → if found, return `shared.ErrLabAlreadyActive`.
    - `inst := lab.New(...)` — state `pending_quota`.
    - `inst.Transition(pending_project, "quota_ok", clock.Now())`.
    - `LabRepo.Create(tx, inst)` — persists row + flushes events to outbox.
    - `proj, err := PoolRepo.AllocateOneFree(tx, kiDomainID, inst.ID)`:
        - On `ErrPoolEmpty` → mark `inst.Transition(rejected, "pool_empty")`, `LabRepo.Save(tx, inst)`, append audit `KindQuotaBlocked` variant, return `ErrPoolEmpty` (tx will commit the rejected state — that's intentional, we want history).
    - `inst.AssignProject(proj.ID, clock.Now())` — transitions to `deploying`.
    - `LabRepo.Save(tx, inst)` and `PoolRepo.Save(tx, proj)` — both flush events.
7. After tx commits:
    - `Queue.Enqueue(TaskDeployLab, payload={lab_id}, IdempotencyKey="deploy:"+labID+":1")`.
    - Return `*lab.LabInstance`.

### `internal/adapters/storage/pgxrepo/quota_cache.go`

Single-row table `quota_cache (id=1, snapshot JSONB, fetched_at)`. Operations:

```go
func (c *QuotaCacheRepo) Read(ctx) (shared.QuotaSnapshot, time.Duration, error)
func (c *QuotaCacheRepo) Write(ctx, snap) error
```

On `Read`, age = `clock.Now() - snapshot.FetchedAt`. If row not found, return `age = math.MaxInt32 seconds` so the caller treats it as stale.

### `internal/app/quota/refresher.go`

asynq handler for `TaskRefreshQuota`:

1. Call `Cloud.GetQuotaSnapshot(ctx)`.
2. `QuotaCache.Write(ctx, snap)`.
3. Done.

The asynq server registers the handler in `cmd/worker/main.go` (follow-up wiring is left as a `// TODO: register here` comment — don't fight cmd/worker for it).

Also schedule a periodic refresh: in a small init in `cmd/worker/main.go`, `asynq.NewScheduler` enqueues `TaskRefreshQuota` every 30s. Add this wiring — it's a 10-line block, well-bounded.

### `internal/adapters/queue/asynq/client.go`

Implement `ports.TaskQueue`:

```go
type Client struct {
    c *asynq.Client
}

func NewClient(addr string) *Client { ... }

func (c *Client) Enqueue(ctx, t ports.Task) (string, error) {
    opts := []asynq.Option{
        asynq.MaxRetry(t.MaxRetries),
        asynq.Queue(queueOf(t.Queue)),
    }
    if t.IdempotencyKey != "" {
        opts = append(opts, asynq.TaskID(t.IdempotencyKey))
    }
    info, err := c.c.EnqueueContext(ctx, asynq.NewTask(string(t.Type), t.Payload), opts...)
    if err != nil { return "", err }
    return info.ID, nil
}

func (c *Client) EnqueueAt(ctx, t, runAt) (string, error) { ... opts with asynq.ProcessAt(runAt) ... }
func (c *Client) Cancel(ctx, taskID) error { ... use asynq.Inspector ... }
```
