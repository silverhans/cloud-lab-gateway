## Critical implementation details

### AllocateOneFree

Use `SELECT ... FOR UPDATE SKIP LOCKED` inside a single SQL statement:

```sql
-- name: AllocateOneFreeProject :one
WITH candidate AS (
    SELECT id FROM projects
    WHERE ki_domain_id = $1 AND state = 'free'
    ORDER BY last_state_change_at
    LIMIT 1
    FOR UPDATE SKIP LOCKED
)
UPDATE projects
SET state = 'allocated',
    allocated_to_lab_id = $2,
    last_state_change_at = now()
WHERE id = (SELECT id FROM candidate)
RETURNING *;
```

The transaction wrapping this query must be the same `Tx` the caller passes (so the lab insertion and the project allocation happen atomically). If no row is returned (pool empty in this domain), return `shared.ErrPoolEmpty`.

### Save with outbox

`Save` runs inside the caller's `Tx`. Steps:

1. `UPDATE projects SET ... WHERE id = $1 AND last_state_change_at = $expected_prev_change_at` — optimistic concurrency check. If `RowsAffected = 0` → `shared.ErrIdempotencyClash`.
2. For each event from `p.PullEvents()`:
   - Insert into `outbox(topic, payload)` with `topic = ev.Kind()`, `payload` = JSON-marshalled struct.
   - Insert into `audit_events(kind, subject_type, subject_id, payload)` for human-readable audit.

Both inserts and the update are in the same transaction → guaranteed atomicity.

### UnitOfWork

```go
type UoW struct{ pool *pgxpool.Pool }

func (u UoW) WithTx(ctx context.Context, fn func(ctx context.Context, tx ports.Tx) error) error {
    return pgx.BeginTxFunc(ctx, u.pool, pgx.TxOptions{IsoLevel: pgx.ReadCommitted}, func(tx pgx.Tx) error {
        return fn(ctx, pgxTx{tx: tx})
    })
}

type pgxTx struct{ tx pgx.Tx }
func (pgxTx) private() {}
```

Repositories accept `ports.Tx`, type-assert to `pgxTx`, use the embedded `pgx.Tx`. Document with a `// unsafe assert ok because UoW is the only constructor of Tx` comment.

### Row → domain mapping

In `pool_mapping.go`:

```go
func projectFromRow(r sqlcgen.Project) *pool.Project {
    var labID *shared.LabInstanceID
    if r.AllocatedToLabID.Valid {
        v := shared.LabInstanceID(r.AllocatedToLabID.UUID)
        labID = &v
    }
    return &pool.Project{
        ID:                shared.ProjectID(r.ID),
        KIProjectID:       r.KiProjectID,
        KIDomainID:        r.KiDomainID,
        Name:              r.Name,
        State:             pool.State(r.State),
        AllocatedToLabID:  labID,
        CleanupFailures:   int(r.CleanupFailures),
        LastStateChangeAt: r.LastStateChangeAt.Time,
        CreatedAt:         r.CreatedAt.Time,
    }
}
```
